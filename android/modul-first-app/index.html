Modul Praktikum Pemrograman Bergerak

Membuat Aplikasi Android Sederhana dengan Input Text dan Button

Tujuan Praktik

  • Memahami penggunaan EditText
  • Memahami penggunaan Button
  • Mengambil input dari user
  • Menampilkan hasil proses ke TextView
  • Menggunakan event klik pada Button

Deskripsi Aplikasi

Aplikasi yang akan dibuat adalah aplikasi Android sederhana yang memungkinkan user menginputkan Nama dan Kota Asal.

Setelah tombol ditekan, aplikasi akan menampilkan sapaan:

Halo Budi dari Mataram, selamat datang!

1. Layout activity_main.xml

Layout menggunakan ConstraintLayout dengan 2 EditText, 1 Button dan 1 TextView.



<?xml version="1.0" encoding="utf-8"?>

<androidx.constraintlayout.widget.ConstraintLayout
xmlns:android="http://schemas.android.com/apk/res/android"
xmlns:app="http://schemas.android.com/apk/res-auto"
android:layout_width="match_parent"
android:layout_height="match_parent">

    <EditText
        android:id="@+id/etNama"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:hint="Masukkan Nama"
        app:layout_constraintTop_toTopOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>

    <EditText
        android:id="@+id/etKota"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:hint="Masukkan Kota Asal"
        app:layout_constraintTop_toBottomOf="@id/etNama"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>

    <Button
        android:id="@+id/btnSapa"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:text="Tampilkan Sapaan"
        app:layout_constraintTop_toBottomOf="@id/etKota"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>

    <TextView
        android:id="@+id/tvHasil"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hasil akan muncul di sini"
        android:textSize="18sp"
        app:layout_constraintTop_toBottomOf="@id/btnSapa"
        app:layout_constraintStart_toStartOf="parent"/>

</androidx.constraintlayout.widget.ConstraintLayout>

2. MainActivity.kt

File ini berfungsi untuk mengambil input user dan menampilkan hasilnya.



import android.os.Bundle
import android.widget.Button
import android.widget.EditText
import android.widget.TextView
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {

    override fun onCreate(savedInstanceState: Bundle?) {

        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        val etNama: EditText = findViewById(R.id.etNama)
        val etKota: EditText = findViewById(R.id.etKota)
        val btnSapa: Button = findViewById(R.id.btnSapa)
        val tvHasil: TextView = findViewById(R.id.tvHasil)

        btnSapa.setOnClickListener {

            val nama = etNama.text.toString()
            val kota = etKota.text.toString()

            if (nama.isNotBlank() && kota.isNotBlank()) {

                val hasil = "Halo $nama dari $kota, selamat datang!"
                tvHasil.text = hasil

            } else {

                tvHasil.text = "Nama dan Kota tidak boleh kosong"

            }

        }

    }

}
