📊 Analisis Histogram

📖 Apa itu Histogram?

Histogram adalah representasi grafis yang menunjukkan distribusi frekuensi nilai intensitas pixel dalam sebuah citra digital. Sumbu horizontal menunjukkan nilai intensitas (0-255 untuk citra 8-bit), sedangkan sumbu vertikal menunjukkan jumlah pixel yang memiliki nilai intensitas tersebut.

🖼️ Input Citra f(x) - 5×5 Pixel

🔄 Operasi Konvolusi

📖 Apa itu Konvolusi?

Konvolusi adalah operasi matematis yang menggeser sebuah kernel (filter) di atas citra untuk menghasilkan citra baru. Setiap pixel hasil dihitung dengan menjumlahkan perkalian antara nilai pixel tetangga dengan bobot kernel.

Rumus: g(x,y) = Σ Σ f(x+i, y+j) × h(i,j)

🖼️ Citra Input f(x) - 5×5

⚙️ Kernel g(x) - 3×3

✨ Hasil Konvolusi - 3×3

* Ukuran output: (5-3+1) × (5-3+1) = 3×3
