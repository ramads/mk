Modul Praktik Pemrograman Bergerak

Membuat Aplikasi Android Sederhana (Input Text dan Button)

Tujuan Praktik

  • Memahami penggunaan EditText pada Android
  • Memahami penggunaan Button
  • Mengambil input dari user
  • Menampilkan hasil proses ke TextView
  • Menggunakan event klik pada button

Deskripsi Aplikasi

Aplikasi yang akan dibuat adalah aplikasi Android sederhana yang memungkinkan user menginputkan nama dan kota asal.

Setelah tombol ditekan, aplikasi akan menampilkan pesan sapaan seperti berikut:

Halo Budi dari Mataram, selamat datang!

MainActivity.kt
activity_main.xml

1. Layout activity_main.xml

Layout ini menggunakan ConstraintLayout dengan komponen:

  • 2 EditText
  • 1 Button
  • 1 TextView


<?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout
xmlns:android="http://schemas.android.com/apk/res/android"
xmlns:app="http://schemas.android.com/apk/res-auto"
android:layout_width="match_parent"
android:layout_height="match_parent">

<EditText
    android:id="@+id/etNama"
    android:layout_width="0dp"
    android:layout_height="wrap_content"
    android:hint="Masukkan Nama"
    app:layout_constraintTop_toTopOf="parent"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"/>

<EditText
    android:id="@+id/etKota"
    android:layout_width="0dp"
    android:layout_height="wrap_content"
    android:hint="Masukkan Kota Asal"
    app:layout_constraintTop_toBottomOf="@id/etNama"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"/>

<Button
    android:id="@+id/btnSapa"
    android:layout_width="0dp"
    android:layout_height="wrap_content"
    android:text="Tampilkan Sapaan"
    app:layout_constraintTop_toBottomOf="@id/etKota"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"/>

<TextView
    android:id="@+id/tvHasil"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hasil akan muncul di sini"
    android:textSize="18sp"
    app:layout_constraintTop_toBottomOf="@id/btnSapa"
    app:layout_constraintStart_toStartOf="parent"/>

</androidx.constraintlayout.widget.ConstraintLayout>

2. MainActivity.kt

Pada file ini kita mengambil input user lalu menampilkannya dalam bentuk teks sapaan. 



import android.os.Bundle
import android.widget.Button
import android.widget.EditText
import android.widget.TextView
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {

override fun onCreate(savedInstanceState: Bundle?) {

super.onCreate(savedInstanceState)
setContentView(R.layout.activity_main)

val etNama: EditText = findViewById(R.id.etNama)
val etKota: EditText = findViewById(R.id.etKota)
val btnSapa: Button = findViewById(R.id.btnSapa)
val tvHasil: TextView = findViewById(R.id.tvHasil)

btnSapa.setOnClickListener {

val nama = etNama.text.toString()
val kota = etKota.text.toString()

if (nama.isNotBlank() && kota.isNotBlank()) {

val hasil = "Halo $nama dari $kota, selamat datang!"
tvHasil.text = hasil

} else {

tvHasil.text = "Nama dan Kota tidak boleh kosong"

}

}

}

}

Alur Program

  1. User mengisi nama
  2. User mengisi kota
  3. User menekan tombol
  4. Aplikasi membaca input
  5. Aplikasi menampilkan sapaan

Latihan Mandiri

  • Field input umur
  • Validasi umur harus angka
  • Tampilan output:
